🚀 Como publicar esta página (GitHub Pages — gratuito)

URL final: https://seu-usuario.github.io/nome-do-repo/

1
Crie uma conta no GitHub
Acesse github.com → Sign up → conta gratuita. Se já tem conta, pule este passo.
2
Crie um repositório novo
Clique em + (canto superior direito) → New repository. Dê o nome que quiser, ex: reuniao-obras. Marque Public. Clique em Create repository.
3
Suba o arquivo HTML
Na página do repositório, clique em uploading an existing file. Arraste o arquivo HTML baixado e renomeie-o para index.html antes de subir. Clique em Commit changes.
4
Ative o GitHub Pages
No repositório: Settings → Pages (menu lateral esquerdo) → em Source selecione Deploy from a branch → Branch: main → pasta / (root) → clique Save.
✓
Aguarde ~1 minuto e acesse o link
O GitHub Pages publica em até 60 segundos. O link aparece na mesma tela Settings → Pages. Formato: https://seu-usuario.github.io/reuniao-obras/. Para atualizar o formulário depois, basta fazer upload do novo index.html no mesmo repositório.
💡 Sobre os dados: O formulário salva automaticamente no navegador (localStorage) — cada computador tem seu próprio rascunho local. Use Exportar JSON para carregar a sessão em outro dispositivo, ou Copiar link para compartilhar o preenchimento atual com alguém via URL.
🔒 Privacidade: O repositório precisa ser Public para o GitHub Pages funcionar no plano gratuito. Se quiser manter o formulário privado, use o plano GitHub Pro (pago) ou opte pelo Netlify, que permite sites privados gratuitamente.
